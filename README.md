# Blog Sederhana

Ini adalah project aplikasi blog sederhana menggunakan PHP Native dan MySQL.

## Fitur

- Menampilkan daftar artikel
- Menambahkan, mengedit, dan menghapus artikel
- Upload gambar untuk artikel
- Login sederhana (opsional)
